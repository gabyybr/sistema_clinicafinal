# ⚠️ Riscos e Mitigação

| Risco                                    | Impacto | Estratégia de Mitigação                              |
| ---------------------------------------- | ------- | ---------------------------------------------------- |
| Falha na integração com o banco de dados | Alta    | Realizar testes de conexão e backups diários         |
| Perda de informações médicas             | Alta    | Utilizar criptografia e rotinas de backup automático |
| Atrasos no desenvolvimento               | Média   | Replanejar atividades e priorizar entregas críticas  |
| Falhas de autenticação                   | Alta    | Implementar validação segura e logs de acesso        |
